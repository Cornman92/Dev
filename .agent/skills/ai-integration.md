### Memory Operations
```bash
# Store important information
memory-store "Project deadline: 2024-03-15, key requirements: scalability, security"

# Retrieve context
memory-recall "project requirements"

# Update knowledge
memory-update "Project deadline extended to 2024-04-01"
```

### Deep Wiki Analysis
```bash
# Analyze repository
deepwiki-analyze "facebook/react" "What are the main architectural patterns?"

# Get documentation
deepwiki-docs "tensorflow/tensorflow" "Installation and setup guide"

# Best practices
deepwiki-ask "microsoft/vscode" "Extension development best practices"
```
