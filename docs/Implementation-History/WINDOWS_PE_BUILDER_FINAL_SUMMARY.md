## 🚀**USAGE EXAMPLES**###**PowerShell Command Line**```powershell

## Launch Ultimate Launcher

.\Scripts\Launch-WindowsPEBuilder.ps1 -Interface GUI

## Create gaming-optimized PE

.\Scripts\Windows-PE-Builder.ps1 -Action Create -PEProfile Gaming
-OutputPath "F:\PE\Gaming-PE" -Architecture x64
-EnableAI $true

## Build comprehensive all-in-one PE

.\Scripts\Windows-PE-Builder.ps1 -Action Build -PEProfile All-in-One
-DriverPath "C:\Drivers" -PluginPath "C:\Plugins"
-SecurityProfile Gaming

## Deploy development PE with network configuration

.\Scripts\Windows-PE-Builder.ps1 -Action Deploy -PEProfile Development
-OutputPath "F:\PE\Dev-PE" -NetworkConfig
"C:\Config\network.xml"

```text

### **GUI Interface**```powershell

## Launch modern GUI

.\Scripts\Windows-PE-Builder-GUI.ps1

```text

### **TUI Interface**```powershell

## Launch modern TUI

.\Scripts\Show-WindowsPEBuilderTUI.ps1

```text

### **Integration Usage**```powershell

## Add PE Builder to Enhanced DISM Suite

.\Scripts\PE-Builder-Integration.ps1 -IntegrationType DISM -Action AddPEBuilder

## Sync data between integrated tools

.\Scripts\PE-Builder-Integration.ps1 -IntegrationType All -Action SyncData

## Test integration

.\Scripts\PE-Builder-Integration.ps1 -IntegrationType All -Action TestIntegration
