# Voice Commands Tutorial

Learn how to master the "Hey C-Man" voice command system

## Setup

1. **Enable Voice Commands**:
   \\\ash
.\GaymerPC\AI-Command-Center\Scripts\Launch-AICommandCenter.ps1 -Mode Voice
  -EnableVoice
   \\\

2.**Test Your Microphone**:

  - Say: "Hey C-Man, hello"
  - Expected response: Time-based greeting

## Basic Commands

### Gaming Commands

- "Hey C-Man, optimize gaming"

- "Hey C-Man, apply competitive profile"

- "Hey C-Man, check performance"

### System Commands

- "Hey C-Man, shutdown computer"

- "Hey C-Man, lock computer"

- "Hey C-Man, check CPU usage"

### AI Assistant

- "Hey C-Man, what time is it"

- "Hey C-Man, help"

- "Hey C-Man, status"

## Tips for Better Recognition

1.**Speak clearly**and at normal volume
2.**Use consistent phrases**from the command list
3.**Ensure quiet environment**for better recognition
4.**Check microphone permissions**in Windows settings
5.**Test regularly**to ensure voice system is working

## Troubleshooting

### Voice Not Working

1. Check microphone permissions
2. Test microphone in Windows settings
3. Restart voice system
4. Check audio drivers

### Commands Not Recognized

1. Use exact phrases from documentation
2. Speak clearly and at normal pace
3. Check for background noise
4. Verify command is supported
