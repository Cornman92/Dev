# Getting Started Tutorial

Welcome to GaymerPC Ultimate Suite! This tutorial will guide you through
the initial setup and basic usage.

## Prerequisites

Before starting, ensure you have:

- Windows 11 Pro x64

- NVIDIA RTX 3060 Ti or equivalent

- 32GB RAM (minimum 16GB)

- Python 3.11+

- PowerShell 7.0+

## Step 1: Installation

\\\ash

## Clone the repository

git clone <<https://github.com/C-Man-Dev/GaymerPC-Suite.git>>
cd GaymerPC-Suite

## Run setup script

.\setup.ps1
\\\

## Step 2: First Launch

\\\ash

## Launch the main interface

\\\ash

## Launch Gaming Suite
